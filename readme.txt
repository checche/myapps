djangoフレームワークを用いて作成しました.

ライブハウスの情報をまとめて確認できるアプリ.

デフォルトでは今後のライブ予定を日付順にリスト表示.

ご挨拶のところから本日のライブ情報一覧を取得可能

右上のテキストボックスにバンド名やライブハウス名を入力することで絞り込みが可能.
好きなバンドの今後の予定が簡単にわかる.

ログイン機能アカウント登録機能を実装
ユーザー情報には名前,メールアドレスの他にお気に入りのバンドが保存される.

マイカレンダー機能(実装予定)
ユーザーのお気に入り情報を取得してお気に入りバンドの出演するライブの日程が目立つように表示される.
複数のお気に入りバンドが共演する場合アプリがそのライブをリコメンドしてくれる.

現状登録ライブハウス数が少ないので増やしていく予定.
