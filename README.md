## 機能
1. トップページで今日以降のライブ情報を表示.
2. ナビバー上の検索欄からライブ情報をバンド名またはライブハウス名で検索.
3. ナビバー上の'バンド'からバンド一覧.また検索欄からバンド名を検索.
4. バンドフォロー機能
5. バンド詳細表示
6. ライブ詳細表示
7. フォローしたバンドの一覧表示
8. フォローしたバンドの出演するライブ一覧表示
9. カレンダー→未実装(カレンダー上にライブ情報が出る予定)
